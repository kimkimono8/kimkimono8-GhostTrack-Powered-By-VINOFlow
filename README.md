# 🛰️ GhostTrack: Powered by VINOFlow

> Ultra-lightweight smart tracking system with OpenVINO + DeepSORT + Super-Resolution.  
> Perfect for home surveillance, RTSP camera monitoring, and embedded AI edge boxes.

---

## 🚀 Features

- 🎯 Real-time Object Detection with YOLOv8 (OpenVINO accelerated)
- 👣 Person + Vehicle Tracking via DeepSORT
- ✨ Enhancement Module: Zero-DCE, CLAHE, Super-Resolution
- 📦 Modular & Scalable Codebase
- ⚡️ Ultra low CPU usage (OpenVINO UHD 620 tested)
